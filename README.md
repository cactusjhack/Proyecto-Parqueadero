🅿️ Mall Plaza Parking System — Domain Model
Academic project developed as part of the Software Development Technology program at ITM (Instituto Tecnológico Metropolitano), Medellín, Colombia.
This repository contains the initial domain model and class architecture for a parking management system designed for the Mall Plaza commercial center.

📐 Architecture Overview
The system is structured around 20+ classes organized into five functional modules:
ModuleResponsibilityPersonsCustomers, employees, and admin rolesVehiclesVehicle types, registration, and categorizationInfrastructureParking spots, zones, and physical layoutOperationsEntry/exit logic, occupancy controlFinancesBilling, rates, and payment records

🛠️ Tech Stack

Language: C#
Platform: .NET 8
IDE: Visual Studio
Design: Object-Oriented Programming (OOP), class modeling


🎯 Learning Goals

Apply OOP principles: encapsulation, inheritance, and abstraction
Design a real-world domain model from business requirements
Structure a multi-module solution in Visual Studio
